## 1. プロダクトビジョン

### Vision
**誰もが制作に挑戦できる世界の実現**

創作活動は本来、すべての人に開かれたものであるべきです。しかし現実には、技術的なスキル・金銭的なコスト・情報へのアクセスといった「環境的ハードル」が、多くのクリエイター志望者の前に立ちはだかっています。私たちは、これらの障壁を取り除き、誰もが自分のアイデアを形にできる世界を目指します。

### Mission
**創作における環境的ハードルを破壊する**

プログラミングができなくてもゲームを作れる。仲間がいなくても相談できる場所がある。技術がなくてもプロのチームに依頼できる。私たちは、プロダクト・コミュニティ・受託制作の3つの事業を通じて、創作の入口から出口までを支える環境を構築します。

### Value
**クリエイターのインナーサークルに居続ける**

クリエイターに寄り添い、彼らの声を聞き、彼らの目線でプロダクトを作る。外側からではなく、創作の現場に身を置き続けることで、真に必要とされるサービスを提供し続けます。

### 創業ストーリー
代表・山縣帆高は、同級生がクリエイターとしての夢を環境的な理由から断念し退学したことをきっかけに起業を決意。「非合理に生きるクリエイター」への敬意を原動力に、創作の環境的ハードルを破壊するための事業を立ち上げました。

## 3. プロダクト一覧

### 3-1. NoLogic（ノーコードゲーム制作プラットフォーム）

| 項目 | 詳細 |
|------|------|
| URL | https://nologic.creatorpot.net/ja / https://nologic.app |
| カテゴリ | SaaS / ノーコードプラットフォーム |
| ステータス | β版（2024年12月リリース） |
| ターゲット | イラストレーター、クリエイター等プログラミング未経験者 |

#### コンセプト
「イラストが描ければ、ゲームが作れる」

NoLogicは、プログラミング知識ゼロでもオリジナルゲームを制作・公開できるノーコードプラットフォームです。クリエイターが持つ最大の資産である「作品（イラスト）」を活かし、ゲームという新しい表現手段を提供します。

#### 制作フロー（3ステップ）
1. **イラストアップロード** - 自分が描いたイラストやキャラクターをアップロード
2. **テンプレート選択** - RPG、パズル、アクション、ノベルなど豊富なジャンルテンプレートから選択
3. **ゲーム完成** - カスタマイズして公開。URLを共有するだけで誰でもプレイ可能

#### 対応ジャンル
- RPG（ロールプレイングゲーム）
- パズルゲーム
- アクションゲーム
- ノベルゲーム
- その他（順次拡大）

#### 差別化ポイント
- **イラストファースト**: プログラミングではなく「絵」を起点としたゲーム制作体験
- **3ステップ完結**: 複雑な開発環境不要。ブラウザだけで完結
- **テンプレート方式**: ゼロから作る必要がなく、選ぶだけでゲームの骨格が完成
- **IPA未踏採択**: 技術力と将来性が国の事業として認められた実績

### 3-2. NextCreators（クリエイターコミュニティ）

| 項目 | 詳細 |
|------|------|
| プラットフォーム | Discord |
| 招待リンク | https://discord.gg/3hy2UNmXca |
| 参加者数 | 100名以上 |
| 参加費 | 無料 |

#### コンセプト
「一人じゃない創作を」

NextCreatorsは、ジャンルを超えたクリエイターが集い、互いに刺激し合えるオンラインコミュニティです。作品の共有、技術的な相談、コラボレーションの機会を提供し、クリエイターの孤独を解消します。

#### 提供価値
- **作品の共有**: 制作途中・完成作品を投稿し、フィードバックを受けられる
- **相談・質問**: 技術的な壁にぶつかったとき、経験者に気軽に相談できる
- **イベント・交流**: 定期的なコミュニティイベント、もくもく会、作品発表会の開催
- **コラボレーション**: 異なるスキルを持つクリエイター同士のマッチング

#### エコシステムとしての位置づけ
NextCreatorsはNoLogicのユーザーコミュニティとしても機能し、制作ノウハウの共有やユーザー同士の交流を促進します。コミュニティで生まれたフィードバックはプロダクト改善に直結し、ユーザードリブンな開発サイクルを実現します。

## 4. バリュープロポジション

### 事業全体のバリュープロポジション
**「創作の入口から出口まで、すべてのハードルを取り除く」**

| ハードル | 解決するプロダクト | 提供価値 |
|----------|-------------------|----------|
| 技術的ハードル | NoLogic | プログラミング不要でゲーム制作を実現 |
| 環境的ハードル | NextCreators | 孤独な制作環境を仲間とのつながりで解消 |
| リソース的ハードル | TechPlant Studio | プロのクリエイターチームが制作を代行 |

### 3事業のシナジー
```
NoLogic（プロダクト）
    ↕ ユーザーフィードバック・コミュニティ連携
NextCreators（コミュニティ）
    ↕ 人材プール・ナレッジ共有
TechPlant Studio（受託制作）
    ↕ 技術力・収益基盤
NoLogic（プロダクト開発に再投資）
```

3つの事業が相互に価値を循環させることで、単独では実現できないエコシステムを構築しています。

## 5. 競合優位性

### NoLogicの競合分析

| 比較軸 | NoLogic | RPGツクール | Unity | Roblox Studio |
|--------|---------|------------|-------|---------------|
| ターゲット | イラストレーター・クリエイター | ゲーム制作者 | ゲーム開発者 | ゲーム開発者 |
| プログラミング | 不要 | 一部必要 | 必須 | 必須 |
| 制作時間 | 数分〜数時間 | 数時間〜数日 | 数週間〜数ヶ月 | 数日〜数週間 |
| イラスト活用 | コア機能 | 素材として利用 | 素材として利用 | 素材として利用 |
| 対応ジャンル | 複数ジャンル | RPG中心 | 全ジャンル | 全ジャンル |
| 学習コスト | 極めて低い | 中程度 | 高い | 中〜高 |

### 独自のポジショニング
- **「イラストレーター向け」という明確なニッチ**: 既存のゲーム制作ツールが「ゲーム開発者」を対象とする中、NoLogicは「絵を描く人」に特化
- **IPA未踏事業採択**: 日本の公的機関から技術的独自性と将来性を認められた唯一のノーコードゲーム制作プラットフォーム
- **コミュニティ一体型開発**: NextCreatorsとの連携により、ユーザーの声をダイレクトにプロダクトに反映できる開発体制
